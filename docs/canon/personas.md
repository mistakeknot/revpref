# RevPref Personas

> Who this product is for, and what hurts for them today.

This is the **lightweight** persona doc produced during onboarding. A deeper Jobs-to-be-Done style interview is tracked as a follow-up bead.

## Personas

### Privacy-Conscious Power User

Runs many desktop apps, launchers, sync tools, and utilities, and wants a faster computer without sending usage history to a cloud service.

### Everyday Desktop User

Uses Windows or macOS for work, games, calls, and browsing, but does not know which old apps are still safe or sensible to remove.

### Open-Source Maintainer

Wants a trustworthy app hygiene tool with inspectable logic, portable data formats, and a contribution path for platform collectors.

## Pain Points

- **Privacy-Conscious Power User:** Existing cleanup tools are either too aggressive, too opaque, or too willing to collect telemetry.
- **Everyday Desktop User:** The installed-app list is long and confusing, and uninstalling the wrong thing feels risky.
- **Open-Source Maintainer:** Cross-platform app inventory and usage tracking are fragmented, hard to test, and easy to make privacy-hostile.
